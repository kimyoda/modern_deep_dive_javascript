# 12.1 함수란?

---

- 자바스크립트에서 가장 중요한 핵심 개념이다.
- 스코프, 실팽텍스트, 클로저, 생성자 함수에 의한 객체 생성, 메서드, this, 르포토 타입, 모듈화 등이 함수와 관련이 있다.
- 프로그래밍 언어의 함수도 수학의 함수와 같은 개념이다.

예제 12-1

```js
// f (x, y) = x + y
function add (x, y) {
    return x + y;
}

// f(2, 5) = 7
add(2, 5); = 7
```

- 함수는 일련의 과정을 문(statement)로 구현, 코드 블록으로 감싸서 하나의 실행 단위로 정의한 것이다.
- 프로그래밍 언어의 함수도 입력을 받아 출력을 내보낸다.
- 함수 내부로 전달 받는 변수를 **매개변수(parameter)**
- **입력을 인수(argument)**, **출력을 반환값(return value)**
- 함수는 값, 여러 개 존재할 수 있으므로 특정 함수를 구별하기 위해 식별자인 함수 이름을 사용할 수 있다.

```js
// 1. 함수 정의 (Function Definition)
function add(x, y) {
  // 함수 이름 add
  // 매개변수 x,y : 함수가 호출될 때 값을 전달받는 변수
  // 함수 몸체: {return x + y}: 함수가 실행할 코드를 담고 있는 블록
  // 반환 값 x+y : return 키워드를 통해 함수 외부로 반환되는 값
  return x + y;
}

// 2. 함수 호출 (Function Call)
// 인수: 2, 5 -> 함수를 호출할 때 매개변수에 실제로 전달하는 값
add(2, 5);
```

예제 12-2

```js
// 함수 정의
fucntion add(x, y) {
    return x + y;
}
```

- 인수를 매개변수를 통해 함수에 전달, 함수의 실행을 명시적으로 지시한다. 이를 함수 호출이라 한다.
- 함수를 호출하면 코드 블록에 담긴 문들이 일괄적으로 실행, 반환값을 반환한다.

예제 12-3

```js
// 함수 호출
var result = add(2, 5);

// 함수 add에 인수 2, 5를 전달, 호출하면 반환값 7을 반환한다.
console.log(result); // 7
```
